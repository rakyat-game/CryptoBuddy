# CryptoBuddy

CryptoBuddy is an app that tracks prices of crypto currencies and allows user's to build their own portfolios

## Planned Features

### Home page
  - show balance
  - search coin
  - watch list
  - open settings page
  - converter for crypto currencies

### Coin Tracker page
  - list crypto currencies 
  - click on crypto currency to show detailed info, like graphs, info text etc.
  -  sort coins by market cap,category, name, etc.
  -  "show more" crypto currencies option
  -  search function

### Portfolio page
  - user balance
  - display graph of user balance
  - show coin holdings
  - add/remove coin function
